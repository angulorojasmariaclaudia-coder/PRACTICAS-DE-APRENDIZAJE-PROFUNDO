# PRACTICAS-DE-APRENDIZAJE-PROFUNDO

Este repositorio contiene el desarrollo de 10 prácticas realizadas durante la asignatura de Aprendizaje Profundo. Donde se abordan distintos tipos de problemas como visión por computador, procesamiento de lenguaje natural y modelos generativos. Cada notebook muestra el proceso completo: desde la preparación de datos hasta el entrenamiento y evaluación de modelos, utilizando librerías como PyTorch, FastAI y Hugging Face Transformers.
